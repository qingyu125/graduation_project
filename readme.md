## 简介
  本仓库为本科毕设代码实现，题目：《面向高效推理的要素抽取与应用算法设计与实现》  
## 整体代码结构  
```
code-enhanced-text-reasoning/  
├── data_processor.py              # DocRED数据处理  
├── text_to_pseudocode.py          # 文本到伪代码转换  
├── pseudocode_parser.py           # 伪代码解析与要素抽取  
├── knowledge_fusion.py            # 知识融合与推理验证  
├── model_training.py              # 模型训练与基线对比  
├── streamlit_app.py               # Streamlit应用系统  
├── requirements.txt               # 依赖包列表  
└── [其他支持文件]  
```
