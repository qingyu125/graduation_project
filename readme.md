## 简介
  本仓库为本科毕设代码实现，题目：《面向高效推理的要素抽取与应用算法设计与实现》  
## 整体代码结构  
```
docred_paper/  
├── train_model.py                     # 模型训练脚本
├── prepare_docred_data                # 数据处理
├── requirements.txt                   # 依赖列表  
├── config/config.yaml                 # 配置文件  
├── README.md                          # 项目说明  
├── data/                              # 数据目录   
├── experiments/                       # 实验输出   
└── src/                               # 源代码  
    ├──enhanced_gui.py                 # 主GUI入口  
    ├── data/                          # 数据处理模块  
    ├── models/                        # 模型管理模块  
    ├── modules/                       # 核心功能模块  
    ├── gui/                           # GUI界面模块  
    └── utils/                         # 工具函数   
```
